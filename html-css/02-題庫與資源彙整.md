# ITS HTML and CSS 證照｜歷屆考題／題庫與資源彙整

> **文件用途**：彙整目前可取得的 ITS HTML and CSS（ITS-301 / INF-301）練習題、題庫、模擬測驗與備考資源，並**標註來源與可信度**，供備考取材參考。
> **調研日期**：2026-07

---

## ⚠️ 先講清楚：關於「歷屆真題」

ITS（Certiport）是**商業國際認證**，官方**不公開歷屆真題**，且考題受版權與保密協議（NDA）保護。因此：

- 網路上流傳、宣稱「XX 題完整真題／verified PDF／100% Pass」的檔案（即 **exam dumps**），**多為考生回憶題或第三方造假題**，其正確性、時效性、合法性都無法保證；**使用 dumps 違反 Certiport NDA，被查獲會撤銷證照**，部分站點還有惡意程式風險，**切勿使用**。
- **最接近「真題手感」的合法管道**是官方授權的 **GMetrix / CertPREP** 與 **MeasureUp / MindHub（INF-301）** 模擬測驗（見下）——由官方技術支援，題型、介面、難度貼近正式考試。
- 本文件將資源分成三級可信度：🟢 官方/授權 → 🟡 教材/課程 → 🟠 社群/回憶題（僅供熟悉題型，勿當標準答案）。

## 五、建議取材優先序（一句話總結）

1. **主軸**：官方 Objective Domains（ITS-301 PDF）+ CodeHS 標準對齊考綱。
2. **練手感**：GMetrix / CertPREP 或 MeasureUp INF-301 官方模擬（至少 1–2 回全真，做到 90% 以上）。
3. **中文教材**：碁峰認證入口 + 天瓏繁中書籍。
4. **補免費**：W3Schools 教學與 Quiz、Quizlet 字卡刷零碎概念（答案需覆核）。
5. **自我檢測**：搭配本專案 `03-模擬題與解析.md`（原創、無版權疑慮）。
6. ❌ **避免**：exam dumps／付費「真題解答」文件站——違反 NDA、會撤照且不可靠。
