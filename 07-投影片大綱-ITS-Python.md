# 第一部分：領域 1
## 資料型別與運算子

---

## W1 引起動機

**提問：** `'5' + 5` 會發生什麼？

- 手機計算機不會出錯，程式卻會 →「型別」概念
- 展示 3 個一行程式的輸出，讓學生先猜

<!-- 用認知衝突開場，比直接講定義更有黏著度。 -->

---

## W1–W2 核心觀念

- 資料型別：`int / float / str / bool / list / dict`
- `type()` 與動態型別；型別轉換 `int() / float() / str()`
- ⚠️ `input()` 一律回傳**字串**
- 運算子優先序：`**` → `* / // %` → `+ -` → 比較 → `and/or/not`
- 字串：索引、切片 `[1:4]`、`[::-1]`、常用方法

<!-- 反覆強調 input() 回傳字串，這是最常見的初學者陷阱與考點。 -->

---

## 課堂活動：型別偵探

兩人一組，判斷 8 個運算式的**型別與數值**，互相對答：

```python
type(7 / 2)      # ?
7 // 2, 7 % 2    # ?
"Py" + "thon"*2  # ?
"Python"[1:4]    # ?
```

<!-- 對應題庫 Q1–Q4，可當場出題。 -->

# 第二部分：領域 2
## 流程控制（拉分關鍵 ⭐）

---

## W3–W4 引起動機

**提問：** 不用複製貼上 100 次，如何印出 1~100？

→ 帶出迴圈的價值

<!-- 領域 2 權重最高最難，這兩週放慢節奏。 -->

---

## W3–W4 核心觀念

- 分支：`if / elif / else`、巢狀、三元運算 `A if C else B`
- 迴圈：`for` + `range(start, stop, step)`（**不含 stop**）
- `while`、`break`、`continue`、`for...else`
- 巢狀迴圈：次數 = 外 × 內

<!-- range 邊界（不含 stop）與 break/continue 差異是高頻考點。 -->

---

## 教學神器：逐行追蹤表

用 [Python Tutor](https://pythontutor.com) 視覺化逐行執行

| 步 | i | 輸出 |
|:--:|:--:|:----:|
| 1 | 0 | 0 |
| 2 | 1 | 1 |
| … | … | … |

<!-- 對迴圈理解最有效的工具，強烈建議實際投影操作。 -->

---

- 進階生：巢狀迴圈印九九乘法表
- 基礎生：單層迴圈累加即可

# 第三部分：領域 3–4
## 輸入輸出 · 函式

---

## W5 輸入輸出與檔案

- `input()`（回傳字串，需轉型）／`print()`（`sep`、`end`）
- 檔案模式：`r`（讀）`w`（覆寫）`a`（附加）
- `with open(...) as f:` 自動關檔
- `read() / readline() / readlines()` 差異

<!-- readlines 回傳串列、readline 回傳單行，常考。 -->

---

## W6 函式與可讀性

- `def`、參數（位置 / 關鍵字 / 預設值 / `*args`）
- `return` 與**作用域**（區域 vs 全域、`global`）
- 無 return 的函式回傳 `None`
- 註解 `#`、docstring `"""..."""`、命名與縮排

<!-- 作用域（Q21, Q90）與「無 return → None」（Q46, Q93）是易錯點。 -->

---

# 第四部分：領域 5–6
## 錯誤處理 · 模組

---

## W7 錯誤處理與模組

- 找錯：語法 / 執行期 / 邏輯錯誤
- `try / except / else / finally`
- 常見例外：`ValueError`、`IndexError`、`KeyError`、`ZeroDivisionError`
- `assert` 與單元測試概念
- 模組：`import`、`math` / `random` / `datetime`

<!-- 領域 5、6 占比低但單純，是投報率高的送分區。 -->

---

## 課堂活動：找錯挑戰賽（W7）

計時，看誰先找出並修正程式碼的錯誤

```python
def divide(a, b):
    return a / b
print(divide(10, 0))   # 哪裡會出錯？如何處理？
```

<!-- 用競賽形式提升參與；引導用 try/except。 -->

## W8 應考策略
