# ITS JavaScript 證照｜模擬題庫擴充（Q27–Q100）

## 領域 1：運算子與最佳實踐

**Q27.** console.log(2 ** 3) 輸出？
- (A) `6`  - (B) `8`  - (C) `9`  - (D) `5`
　→ **答案：B**　** 是次方：2 的 3 次方 = 8。

**Q28.** console.log(7 / 2) 輸出？
- (A) `3`  - (B) `3.5`  - (C) `3.0`  - (D) `4`
　→ **答案：B**　JS 的 / 是浮點除法：7/2 = 3.5。

**Q29.** console.log("a" + 1 + 2) 輸出？
- (A) `a3`  - (B) `a12`  - (C) `3a`  - (D) `a21`
　→ **答案：B**　由左而右：'a'+1='a1'，再 +2='a12'（字串串接）。

**Q30.** console.log(1 + 2 + "3") 輸出？
- (A) `33`  - (B) `123`  - (C) `6`  - (D) `15`
　→ **答案：A**　由左而右：1+2=3（數字），再 +'3'='33'。

**Q31.** console.log(typeof 42) 輸出？
- (A) `int`  - (B) `number`  - (C) `Number`  - (D) `42`
　→ **答案：B**　JS 數字型別為 'number'（不分整數/浮點）。

**Q32.** console.log(typeof true) 輸出？
- (A) `bool`  - (B) `boolean`  - (C) `Boolean`  - (D) `true`
　→ **答案：B**　布林型別為 'boolean'。

**Q33.** console.log(typeof undefined) 輸出？
- (A) `null`  - (B) `undefined`  - (C) `object`  - (D) `undef`
　→ **答案：B**　未定義值的型別為 'undefined'。

**Q34.** console.log(10 === 10, 10 === "10") 輸出？
- (A) `true true`  - (B) `true false`  - (C) `false false`  - (D) `false true`
　→ **答案：B**　=== 需值與型別皆同；10 與字串 '10' 型別不同 → false。

**Q35.** console.log(!true) 輸出？
- (A) `true`  - (B) `false`  - (C) `1`  - (D) `undefined`
　→ **答案：B**　! 是邏輯反轉：!true = false。

**Q36.** console.log(true && false) 輸出？
- (A) `true`  - (B) `false`  - (C) `undefined`  - (D) `錯誤`
　→ **答案：B**　&& 兩者皆真才真；含 false → false。

**Q37.** console.log(false || "hi") 輸出？
- (A) `false`  - (B) `hi`  - (C) `true`  - (D) `undefined`
　→ **答案：B**　|| 回傳第一個為真的值；false 為假 → 回傳 'hi'。

**Q38.** let x = 5；x++；console.log(x) 輸出？
- (A) `5`  - (B) `6`  - (C) `4`  - (D) `55`
　→ **答案：B**　x++ 遞增 1 → 6。

**Q39.** console.log(Boolean(0), Boolean("a")) 輸出？
- (A) `false true`  - (B) `true false`  - (C) `false false`  - (D) `true true`
　→ **答案：A**　0 為假值；非空字串 'a' 為真值。

**Q40.** 輸出（兩行）為何？
```javascript
try {
  throw new Error("x");
} catch (e) {
  console.log("A");
} finally {
  console.log("B");
}
```
- (A) `A 然後 B`  - (B) `只有 A`  - (C) `只有 B`  - (D) `B 然後 A`
　→ **答案：A**　throw 觸發 catch 印 A；finally 一定執行印 B。

## 領域 2：資料型別與函式

**Q41.** let a = [1,2,3]；a.push(4)；console.log(a.length) 輸出？
- (A) `3`  - (B) `4`  - (C) `5`  - (D) `1`
　→ **答案：B**　push 在尾端加一個元素，長度變 4。

**Q42.** console.log([3, 1, 2].sort()) 輸出？
- (A) `[3, 1, 2]`  - (B) `[1, 2, 3]`  - (C) `[3, 2, 1]`  - (D) `[1, 3, 2]`
　→ **答案：B**　sort() 預設由小到大排序。

**Q43.** console.log(["b", "a"].join("-")) 輸出？
- (A) `a-b`  - (B) `b-a`  - (C) `ba`  - (D) `b,a`
　→ **答案：B**　join 依原順序以 '-' 串接（不會排序）→ 'b-a'。

**Q44.** console.log([1, 2, 3].reverse()) 輸出？
- (A) `[1, 2, 3]`  - (B) `[3, 2, 1]`  - (C) `[2, 1, 3]`  - (D) `錯誤`
　→ **答案：B**　reverse 反轉陣列順序。

**Q45.** console.log("hello".length) 輸出？
- (A) `4`  - (B) `5`  - (C) `6`  - (D) `undefined`
　→ **答案：B**　字串長度為 5。

**Q46.** console.log("hello".toUpperCase()) 輸出？
- (A) `hello`  - (B) `HELLO`  - (C) `Hello`  - (D) `錯誤`
　→ **答案：B**　toUpperCase 轉大寫。

**Q47.** console.log("a,b,c".split(",").length) 輸出？
- (A) `1`  - (B) `3`  - (C) `5`  - (D) `2`
　→ **答案：B**　split(',') 切成 ['a','b','c']，長度 3。

**Q48.** console.log(Math.round(4.5)) 輸出？
- (A) `4`  - (B) `5`  - (C) `4.5`  - (D) `6`
　→ **答案：B**　round 四捨五入：4.5 → 5。

**Q49.** console.log(Math.ceil(4.1)) 輸出？
- (A) `4`  - (B) `5`  - (C) `4.1`  - (D) `6`
　→ **答案：B**　ceil 向上取整 → 5。

**Q50.** console.log(Math.abs(-9)) 輸出？
- (A) `-9`  - (B) `9`  - (C) `0`  - (D) `錯誤`
　→ **答案：B**　abs 取絕對值 → 9。

**Q51.** 輸出為何？
```javascript
const f = function(x) {
  return x * x;
};
console.log(f(4));
```
- (A) `8`  - (B) `16`  - (C) `4`  - (D) `錯誤`
　→ **答案：B**　函式表達式：4×4=16。

**Q52.** 輸出為何？
```javascript
const g = (a, b) => a + b;
console.log(g(3, 4));
```
- (A) `34`  - (B) `7`  - (C) `12`  - (D) `錯誤`
　→ **答案：B**　箭頭函式回傳 a+b=7。

**Q53.** 輸出為何？
```javascript
function h(a, b = 10) {
  return a + b;
}
console.log(h(5));
```
- (A) `5`  - (B) `15`  - (C) `錯誤`  - (D) `NaN`
　→ **答案：B**　b 用預設值 10 → 5+10=15。

**Q54.** const o = {x:1, y:2}；console.log(Object.keys(o).length) 輸出？
- (A) `1`  - (B) `2`  - (C) `3`  - (D) `undefined`
　→ **答案：B**　物件有兩個鍵 x、y，長度 2。

## 領域 3：流程控制與邏輯

**Q55.** 輸出為何？
```javascript
let x = 8, r;
if (x < 5) r = "low";
else if (x < 10) r = "mid";
else r = "high";
console.log(r);
```
- (A) `low`  - (B) `mid`  - (C) `high`  - (D) `undefined`
　→ **答案：B**　8 不 <5，但 <10 → 'mid'。

**Q56.** 輸出為何？
```javascript
let sum = 0;
const a = [10, 20, 30];
for (let i = 0; i < a.length; i++) sum += a[i];
console.log(sum);
```
- (A) `30`  - (B) `60`  - (C) `0`  - (D) `102030`
　→ **答案：B**　10+20+30=60。

**Q57.** 輸出為何？
```javascript
let i = 0, c = 0;
while (i < 5) {
  c++;
  i++;
}
console.log(c);
```
- (A) `4`  - (B) `5`  - (C) `6`  - (D) `0`
　→ **答案：B**　i 從 0 到 4 共執行 5 次。

**Q58.** 輸出為何？
```javascript
let n = 0, out = 0;
do {
  out++;
  n++;
} while (n < 3);
console.log(out);
```
- (A) `2`  - (B) `3`  - (C) `4`  - (D) `0`
　→ **答案：B**　do-while 至少執行一次，n=0,1,2 共 3 次。

**Q59.** 輸出為何？
```javascript
let s = "";
for (const ch of "abc") s += ch + ch;
console.log(s);
```
- (A) `abc`  - (B) `aabbcc`  - (C) `abcabc`  - (D) `aaabbbccc`
　→ **答案：B**　每個字元重複一次：a→aa、b→bb、c→cc。

**Q60.** 輸出為何？
```javascript
let r = 0;
for (let i = 0; i < 10; i++) {
  if (i === 3) break;
  r++;
}
console.log(r);
```
- (A) `3`  - (B) `4`  - (C) `10`  - (D) `9`
　→ **答案：A**　i=3 時 break，r 只累加 i=0,1,2 三次。

**Q61.** 輸出為何？
```javascript
let sum = 0;
for (let i = 1; i <= 5; i++) {
  if (i % 2 === 0) continue;
  sum += i;
}
console.log(sum);
```
- (A) `9`  - (B) `15`  - (C) `6`  - (D) `5`
　→ **答案：A**　只加奇數 1+3+5=9（偶數被 continue 跳過）。

**Q62.** 輸出為何？
```javascript
let c = 0;
for (let i = 0; i < 3; i++)
  for (let j = 0; j < 3; j++)
    c++;
console.log(c);
```
- (A) `3`  - (B) `6`  - (C) `9`  - (D) `27`
　→ **答案：C**　3×3=9 次。

**Q63.** let age = 20；console.log(age >= 18 ? "adult" : "minor") 輸出？
- (A) `adult`  - (B) `minor`  - (C) `true`  - (D) `20`
　→ **答案：A**　三元運算：20>=18 為真 → 'adult'。

**Q64.** 輸出為何？（注意 case 未 break）
```javascript
let x = 1, o = "";
switch (x) {
  case 1: o += "A";
  case 2: o += "B"; break;
  case 3: o += "C";
}
console.log(o);
```
- (A) `A`  - (B) `AB`  - (C) `ABC`  - (D) `B`
　→ **答案：B**　case 1 沒 break 會往下執行 case 2 → 'AB'。

**Q65.** 輸出為何？
```javascript
let a = 0;
let b = a || 5;
console.log(b);
```
- (A) `0`  - (B) `5`  - (C) `true`  - (D) `undefined`
　→ **答案：B**　|| 前者 0 為假值 → 回傳後者 5。

**Q66.** console.log(3 < 5 && 5 < 10) 輸出？
- (A) `true`  - (B) `false`  - (C) `3`  - (D) `錯誤`
　→ **答案：A**　兩個比較皆為真，&& → true。

**Q67.** 輸出為何？
```javascript
const nums = [2, 4, 6];
let t = 1;
for (const n of nums) t *= n;
console.log(t);
```
- (A) `12`  - (B) `48`  - (C) `246`  - (D) `0`
　→ **答案：B**　2×4×6=48。

**Q68.** 輸出為何？
```javascript
let count = 0;
for (const ch of "banana") if (ch === "a") count++;
console.log(count);
```
- (A) `2`  - (B) `3`  - (C) `1`  - (D) `6`
　→ **答案：B**　banana 有 3 個 a。

**Q69.** 輸出為何？
```javascript
let out = "";
for (let i = 3; i > 0; i--) out += i;
console.log(out);
```
- (A) `123`  - (B) `321`  - (C) `3210`  - (D) `3`
　→ **答案：B**　從 3 遞減到 1：'321'。

**Q70.** 輸出為何？
```javascript
let even = 0;
for (let i = 1; i <= 10; i++) if (i % 2 === 0) even++;
console.log(even);
```
- (A) `4`  - (B) `5`  - (C) `6`  - (D) `10`
　→ **答案：B**　2,4,6,8,10 共 5 個偶數。

## 領域 4：DOM 與事件

**Q71.** 選取第一個符合 CSS 選擇器 .item 的元素，用？
- (A) `document.querySelector(".item")`  - (B) `document.querySelectorAll(".item")`  - (C) `document.getElementById(".item")`  - (D) `document.select(".item")`
　→ **答案：A**　querySelector 回傳第一個符合的元素。

**Q72.** 選取所有符合 .item 的元素（NodeList），用？
- (A) `document.querySelector(".item")`  - (B) `document.querySelectorAll(".item")`  - (C) `document.getElement(".item")`  - (D) `document.all(".item")`
　→ **答案：B**　querySelectorAll 回傳所有符合的元素集合。

**Q73.** 建立一個新的 div 元素，用？
- (A) `document.newElement("div")`  - (B) `document.createElement("div")`  - (C) `document.addElement("div")`  - (D) `new Element("div")`
　→ **答案：B**　document.createElement 建立新元素。

**Q74.** 把子節點加到父元素的末端，用？
- (A) `parent.appendChild(child)`  - (B) `parent.addChild(child)`  - (C) `parent.append(child, true)`  - (D) `child.appendTo(parent)`
　→ **答案：A**　appendChild 把節點加到父元素子節點末端。

**Q75.** 移除某個子節點，用？
- (A) `parent.deleteChild(child)`  - (B) `parent.removeChild(child)`  - (C) `child.remove(parent)`  - (D) `parent.delete(child)`
　→ **答案：B**　removeChild 從父元素移除指定子節點。

**Q76.** 設定元素的 id 屬性，用？
- (A) `element.setAttribute("id", "x")`  - (B) `element.attr("id", "x")`  - (C) `element.set("id", "x")`  - (D) `element.addAttr("id", "x")`
　→ **答案：A**　setAttribute(名稱, 值) 設定屬性。

**Q77.** 讀取元素的 id 屬性值，用？
- (A) `element.getAttribute("id")`  - (B) `element.attr("id")`  - (C) `element.readAttribute("id")`  - (D) `element.get("id")`
　→ **答案：A**　getAttribute(名稱) 讀取屬性值。

**Q78.** 為元素加一個 CSS class "active"，用？
- (A) `element.class += "active"`  - (B) `element.classList.add("active")`  - (C) `element.addClass("active")`  - (D) `element.class.add("active")`
　→ **答案：B**　classList.add 新增 class。

**Q79.** 只設定純文字（不解析 HTML 標籤），用哪個屬性？
- (A) `innerHTML`  - (B) `textContent`  - (C) `innerHTML 才安全`  - (D) `value`
　→ **答案：B**　textContent 只設純文字；innerHTML 會解析 HTML。

**Q80.** 事件處理函式中，代表「觸發事件的元素」的是？
- (A) `event.source`  - (B) `event.target`  - (C) `event.element`  - (D) `this.event`
　→ **答案：B**　event.target 是實際觸發事件的元素。

**Q81.** 要為同一元素綁定「多個」click 處理函式，應用？
- (A) `onclick 屬性`  - (B) `addEventListener`  - (C) `element.click()`  - (D) `onclick 可重複指定`
　→ **答案：B**　addEventListener 可綁多個；onclick 屬性只會保留最後一個。

**Q82.** 要等整份 HTML 的 DOM 載入完成再執行程式，監聽哪個事件？
- (A) `load`  - (B) `DOMContentLoaded`  - (C) `ready`  - (D) `start`
　→ **答案：B**　DOMContentLoaded 在 DOM 解析完成時觸發（早於圖片等資源的 load）。

**Q83.** 取得某元素底下所有子元素集合，用哪個屬性？
- (A) `element.children`  - (B) `element.nodes`  - (C) `element.sub`  - (D) `element.child`
　→ **答案：A**　element.children 回傳子元素集合。

**Q84.** 依 class 名稱取得元素集合，用？
- (A) `document.getElementsByClassName("x")`  - (B) `document.getClass("x")`  - (C) `document.getElementByClass("x")`  - (D) `document.className("x")`
　→ **答案：A**　getElementsByClassName 依 class 取得元素集合。

## 領域 5：表單與輸入

**Q85.** 取得 checkbox 是否被勾選，讀哪個屬性？
- (A) `checkbox.value`  - (B) `checkbox.checked`  - (C) `checkbox.selected`  - (D) `checkbox.on`
　→ **答案：B**　checked 為布林值，表示是否勾選。

**Q86.** 取得下拉選單 select 目前選取的值，用？
- (A) `select.value`  - (B) `select.text`  - (C) `select.option`  - (D) `select.selected`
　→ **答案：A**　select.value 是目前選項的 value。

**Q87.** 取得下拉選單目前選項的索引，用？
- (A) `select.index`  - (B) `select.selectedIndex`  - (C) `select.currentIndex`  - (D) `select.pos`
　→ **答案：B**　selectedIndex 是目前選項的索引。

**Q88.** 使用者送出表單時觸發的事件是？
- (A) `click`  - (B) `submit`  - (C) `send`  - (D) `change`
　→ **答案：B**　表單送出觸發 submit 事件。

**Q89.** 讓輸入欄位成為必填的 HTML 屬性是？
- (A) `needed`  - (B) `required`  - (C) `mustfill`  - (D) `validate`
　→ **答案：B**　required 屬性使欄位必填。

**Q90.** 事件處理中取得觸發輸入框目前的值，用？
- (A) `event.value`  - (B) `event.target.value`  - (C) `event.input`  - (D) `this.value.get()`
　→ **答案：B**　event.target 是該元素，.value 取其值。

**Q91.** 用 JavaScript 以程式方式送出表單，用？
- (A) `form.send()`  - (B) `form.submit()`  - (C) `form.post()`  - (D) `form.go()`
　→ **答案：B**　form.submit() 送出表單。

**Q92.** 清空/重設表單所有欄位，用？
- (A) `form.clear()`  - (B) `form.reset()`  - (C) `form.empty()`  - (D) `form.default()`
　→ **答案：B**　form.reset() 重設欄位為初始值。

**Q93.** 讓某個輸入框自動取得游標焦點，用？
- (A) `input.focus()`  - (B) `input.active()`  - (C) `input.select()`  - (D) `input.cursor()`
　→ **答案：A**　input.focus() 使欄位取得焦點。

**Q94.** 一組同名 radio 中，如何判斷某顆是否被選？
- (A) `radio.value`  - (B) `radio.checked`  - (C) `radio.selected`  - (D) `radio.on`
　→ **答案：B**　用被選那顆的 checked 為 true 判斷。

**Q95.** 表單資料附在網址查詢字串送出的 method 是？
- (A) `GET`  - (B) `POST`  - (C) `PUT`  - (D) `SEND`
　→ **答案：A**　GET 把資料放 URL 查詢字串。

**Q96.** 表單資料放在請求主體、較適合大量或敏感資料的 method 是？
- (A) `GET`  - (B) `POST`  - (C) `HEAD`  - (D) `VIEW`
　→ **答案：B**　POST 放請求主體，不顯示於網址。

**Q97.** 指定表單送出目的地網址的屬性是？
- (A) `action`  - (B) `href`  - (C) `target`  - (D) `src`
　→ **答案：A**　form 的 action 屬性指定送出目的地。

**Q98.** 設定輸入框的提示文字（灰字），用哪個屬性？
- (A) `hint`  - (B) `placeholder`  - (C) `tip`  - (D) `label`
　→ **答案：B**　placeholder 顯示提示文字。

**Q99.** 讓瀏覽器內建驗證 email 格式，input 的 type 應設為？
- (A) `type="text"`  - (B) `type="email"`  - (C) `type="mail"`  - (D) `type="string"`
　→ **答案：B**　type="email" 觸發 HTML5 內建 email 驗證。

**Q100.** 表單中負責觸發送出的按鈕，type 應為？
- (A) `"button"`  - (B) `"submit"`  - (C) `"send"`  - (D) `"form"`
　→ **答案：B**　type="submit" 的按鈕會送出表單。
