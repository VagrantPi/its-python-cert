# ITS JavaScript 證照｜歷屆考題／題庫與資源彙整

> **文件用途**：彙整目前網路上可取得的 ITS JavaScript（INF-302）練習題、題庫、模擬測驗與備考資源，並**標註來源與可信度**，供備考取材參考。
> **調研日期**：2026-07

---

## ⚠️ 先講清楚：關於「歷屆真題」

ITS（Certiport）是**商業國際認證**，官方**不公開歷屆真題**，且考題受版權與保密協議（NDA）保護。因此：

- 網路上宣稱「INF-302 dumps／完整真題／100% Pass」的檔案（如 marks4sure、各 dumps 站），**多為第三方回憶題或拼湊題**，正確性、時效性、合法性都無法保證，且可能違反考試誠信，**不建議使用**。
- **最接近真題手感的合法管道**是官方授權的 **GMetrix / CertPREP / MeasureUp** 模擬測驗——由 Certiport 官方技術支援，題型、介面、難度貼近正式考試。
- 三級可信度：🟢 官方/授權 → 🟡 教材/課程 → 🟠 社群/回憶題（僅供熟悉題型，勿當標準答案）。

## 五、建議取材優先序（一句話總結）

1. **主軸**：中文教材（含模擬題）+ CodeHS 21 條標準對齊考綱。
2. **練手感**：GMetrix / CertPREP / MeasureUp 官方模擬（至少 1–2 回全真）。
3. **補免費**：Quizlet 字卡刷零碎概念。
4. **自我檢測**：搭配本資料夾 `03-模擬題與解析.md`（原創、Node.js 實跑驗證、無版權疑慮）。
5. ❌ 避免：INF-302「dumps／真題解答」販售站——侵權且不可靠。
