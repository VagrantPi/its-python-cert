Overview

這張證照在考什麼

ITS JavaScript（考科 INF-302，前身 Microsoft 98-382）屬入門級，重點在讀懂、除錯與網頁互動（DOM／事件／表單）。題型包含單選、複選、拖曳、排序、下拉等；台灣由碁峰資訊總代理，考場提供中文介面。

⚠️ 官方不公開歷屆真題（受 NDA 保護）。最接近真題手感的合法管道是官方授權的 GMetrix / CertPREP / MeasureUp 模擬；本站 100 題為原創、語法題經 Node.js 實跑驗證，無版權疑慮。
