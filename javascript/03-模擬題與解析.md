## 領域 1：運算子與最佳實踐（Q1–Q6）

**Q1.** `console.log(typeof "hello")` 輸出？
- (A) `string` (B) `"hello"` (C) `text` (D) `String`

**Q2.** `console.log(5 + "5")` 輸出？
- (A) `10` (B) `55` (C) `"10"` (D) 錯誤

**Q3.** `console.log(5 == "5", 5 === "5")` 輸出？
- (A) `true true` (B) `true false` (C) `false false` (D) `false true`

**Q4.** `console.log(10 % 3)` 輸出？
- (A) `3` (B) `1` (C) `0` (D) `3.33`

**Q5.** 以下程式輸出？
```javascript
try {
  let x = null;
  x.foo;
} catch (e) {
  console.log("caught");
}
```
- (A) `null` (B) `caught` (C) `undefined` (D) 程式崩潰

**Q6.** `console.log(typeof null)` 輸出？（JS 經典特性）
- (A) `null` (B) `object` (C) `undefined` (D) `false`

## 領域 2：資料型別與函式（Q7–Q12）

**Q7.** `let arr = [1, 2, 3]; console.log(arr.length)` 輸出？
- (A) `2` (B) `3` (C) `4` (D) `undefined`

**Q8.** `console.log([1, 2, 3].indexOf(2))` 輸出？
- (A) `0` (B) `1` (C) `2` (D) `-1`

**Q9.** `console.log(Math.max(3, 7, 2))` 輸出？
- (A) `2` (B) `3` (C) `7` (D) `12`

**Q10.** `console.log(Math.floor(4.7))` 輸出？
- (A) `4` (B) `5` (C) `4.7` (D) `5.0`

**Q11.** 以下程式輸出？
```javascript
function add(a, b) {
  return a + b;
}
console.log(add(2, 3));
```
- (A) `23` (B) `5` (C) `undefined` (D) 錯誤

**Q12.** `const obj = { name: "Amy" }; console.log(obj.name)` 輸出？
- (A) `name` (B) `Amy` (C) `undefined` (D) 錯誤

## 領域 3：流程控制與邏輯（Q13–Q16）

**Q13.** `let x = 5; console.log(x > 3 ? "big" : "small")` 輸出？
- (A) `big` (B) `small` (C) `true` (D) `5`

**Q14.** 以下程式輸出？
```javascript
let s = 0;
for (let i = 1; i <= 4; i++) {
  s += i;
}
console.log(s);
```
- (A) `4` (B) `10` (C) `24` (D) `15`

**Q15.** 以下程式輸出？
```javascript
let d = 2;
switch (d) {
  case 1: console.log("A"); break;
  case 2: console.log("B"); break;
  default: console.log("C");
}
```
- (A) `A` (B) `B` (C) `C` (D) `A B C`

**Q16.** 以下程式輸出？
```javascript
let n = 3, r = 1;
while (n > 0) {
  r *= n;
  n--;
}
console.log(r);
```
- (A) `3` (B) `6` (C) `9` (D) `0`

**Q18.** 要**同時**設定元素的文字與其中的 HTML 標籤，用哪個屬性？

**Q21.** `document.write("Hi")` 的作用是？

## 📖 解析與答案（Answer Key）

> 標註 ⚙️ 者為 Node.js 實跑驗證的輸出；其餘為 DOM/事件/表單觀念題（依官方語法/MDN）。

| 題 | 答案 | 解析 |
|:--:|:----:|------|
| Q1 | **A** | ⚙️ `typeof` 回傳型別字串，字串的型別是 `"string"`。 |
| Q2 | **B** | ⚙️ `+` 遇字串會**字串串接**：數字 5 被轉成字串 → `"55"`（不是相加）。 |
| Q3 | **B** | ⚙️ `==` 會型別轉換後比較（5 == "5" 為 true）；`===` 嚴格相等需型別也相同（false）。 |
| Q4 | **B** | ⚙️ `%` 取餘數：10 ÷ 3 餘 1。 |
| Q5 | **B** | ⚙️ 存取 `null` 的屬性會拋 `TypeError`，被 `catch` 攔截 → 印 `caught`。 |
| Q6 | **B** | ⚙️ JS 經典特性：`typeof null === "object"`（歷史遺留 bug，考試常考）。 |
| Q7 | **B** | ⚙️ 陣列有 3 個元素，`length` 為 3。 |
| Q8 | **B** | ⚙️ `indexOf(2)` 回傳值 2 的索引 1（找不到才回 -1）。 |
| Q9 | **C** | ⚙️ `Math.max` 回傳最大值 7。 |
| Q10 | **A** | ⚙️ `Math.floor` 無條件捨去（向下取整）→ 4。 |
| Q11 | **B** | ⚙️ 兩個數字相加，`return 2+3` → 5。 |
| Q12 | **B** | ⚙️ 以 `.` 存取物件屬性 `name` → `Amy`。 |
| Q13 | **A** | ⚙️ 三元運算：`x>3` 為真 → `big`。 |
| Q14 | **B** | ⚙️ 1+2+3+4 = 10。 |
| Q15 | **B** | ⚙️ `d===2` 命中 `case 2` → 印 `B`（有 `break` 不會往下）。 |
| Q16 | **B** | ⚙️ 3×2×1 = 6（while 迴圈算階乘）。 |
| Q17 | **A** | `document.getElementById("id")` 依 id 取單一元素（注意不加 `#`）。 |
| Q18 | **C** | `innerHTML` 可設定含 HTML 標籤的內容；`textContent` 只設純文字。 |
| Q19 | **B** | 標準做法 `addEventListener("click", 處理函式)`。 |
| Q20 | **A** | 鍵盤按下為 `keydown`（放開為 `keyup`）。 |
| Q21 | **B** | `document.write()` 把內容寫進 HTML 文件輸出流。 |
| Q22 | **B** | 透過 `element.style.backgroundColor` 改樣式（CSS 的 `background-color` 對應駝峰 `backgroundColor`）。 |
| Q23 | **B** | 表單欄位用 `.value` 取／設值。 |
| Q24 | **B** | `event.preventDefault()` 阻止預設行為（如表單送出、連結跳轉）。 |
| Q25 | **B** | 檢查欄位內容合法與否＝**輸入驗證**。 |
| Q26 | **B** | `POST` 把資料放在請求主體，不顯示於網址；`GET` 會附在 URL 查詢字串。 |
