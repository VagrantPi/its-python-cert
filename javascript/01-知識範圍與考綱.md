> **證照全名**：Certiport IT Specialist (ITS) — JavaScript
> **考科代碼**：**INF-302**（前身為 ITS-302；更早為 Microsoft **98-382 Introduction to Programming Using JavaScript**）
> **發行單位**：Certiport（Pearson VUE 旗下）
> **台灣總代理**：碁峰資訊
> **文件用途**：完整梳理考試涵蓋的知識範圍、五大領域權重與細部技能點，作為備考的主幹地圖。
> **調研日期**：2026-07（以官方 Objective Domains、CodeHS 標準框架與多方來源交叉驗證）

## 一、證照定位

ITS（IT Specialist）JavaScript 是 Certiport 針對**入門級／基礎程度**設計的國際認證，用來證明應考者能：

> 「辨識、撰寫並除錯能以邏輯方式解決問題的 JavaScript 程式碼。」
> （*recognize, write, and debug JavaScript code that will logically solve a problem*）

- **難度定位**：入門～基礎。重點在**讀懂程式、理解語法、DOM/事件操作與找錯**，而非在考場開發完整應用。
- **與 Python 版差異**：ITS JavaScript **多了「網頁」面向**——DOM（文件物件模型）、事件處理、HTML 表單與驗證，這是它和 ITS Python 最大的不同（Python 版沒有這三塊，改為模組與檔案 I/O）。
- **適合對象**：高中職／大專與成人網頁程式初學者、想要一張國際入門證照豐富學習歷程或履歷者。
- **上游關係**：承接自 Microsoft **98-382**，範圍高度重疊，舊教材可參考。

## 三、五大考試領域與權重（Objective Domains）

官方 Objective Domains 將考試分為 5 大領域、共 21 項子目標（見下）。各領域**權重為近似值**，依 INF-302 官方練習題庫（共 138 題）的題數分布推得，與官方藍圖比例一致。

| # | 領域（Domain） | 近似權重 | 對應主題 |
|---|----------------|:--------:|---------|
| 1 | Operators and Best Practices（運算子與最佳實踐） | **≈ 21%** | 運算子、最佳實踐、內外部腳本、例外處理、BOM |
| 2 | Data Types and Functions（資料型別與函式） | **≈ 23%** | 變數與原始型別、陣列、物件、Math、函式 |
| 3 | Control Flow and Logic（流程控制與邏輯） | **≈ 14%** | 邏輯/比較運算、判斷式、迴圈 |
| 4 | DOM and Events（文件物件模型與事件） | **≈ 23%** | DOM 樹、事件處理、輸出到 HTML、增修元素、監聽器 |
| 5 | Forms and Input（表單與輸入） | **≈ 18%** | 取值/設值、輸入驗證、表單送出流程 |

> 📌 **備考策略提示**：**領域 2 與 4 各約 23%、合計近半**，是必守重心；領域 4（DOM/事件）與領域 5（表單）是 JavaScript 版**特有的網頁互動**考點，最容易和只會「純語法」的考生拉開差距。領域 3 權重最低但概念單純，屬送分區。

## 四、各領域細部技能點（21 項官方子目標）

> 以下 21 項對應官方 Objective Domains / CodeHS 標準框架，是最權威的「考什麼」清單。

### 領域 1：運算子與最佳實踐（1.1–1.5）
- **1.1** 使用**指派與算術運算子**完成並除錯程式
  → `= + - * / % ** ++ --`、複合指派 `+= -=`、字串串接 `+`（注意 `5 + "5"` = `"55"`）。
- **1.2** 套用 **JavaScript 最佳實踐**
  → `var`/`let`/`const` 選用、命名、分號、嚴格相等 `===`、避免全域污染、註解。
- **1.3** 判斷**內部 vs 外部腳本**
  → `<script>` 內嵌與 `<script src="...">` 外部引用、放置位置、`defer`/`async`。
- **1.4** 實作**例外處理**
  → `try / catch / finally`、`throw`、`Error` 物件、語法錯誤 vs 執行期錯誤。
- **1.5** 除錯與 **BOM（瀏覽器物件模型）** 互動的程式
  → `window`、`alert()`/`prompt()`/`confirm()`、`window.open/close`、`screen`、`console.log`。

### 領域 2：資料型別與函式（2.1–2.5）
- **2.1** 宣告與使用**原始型別變數**
  → `number`、`string`、`boolean`、`undefined`、`null`、`typeof`（注意 `typeof null === "object"`）、型別轉換 `Number()/String()/parseInt()/parseFloat()`。
- **2.2** 宣告與使用**陣列**
  → 建立、索引、`length`、`push/pop/shift/unshift`、`indexOf`、`sort`、多維陣列。
- **2.3** 完成並除錯**物件**相關程式
  → 物件字面值、屬性存取（`.` 與 `[]`）、方法、`String` 物件、`Date` 物件、`forEach`。
- **2.4** 除錯使用**內建 Math 函式**的程式
  → `Math.max/min`、`Math.floor/ceil/round`、`Math.abs`、`Math.random`、`Math.pow/sqrt`、`Math.PI`。
- **2.5** 除錯**含參數與回傳值的函式**
  → 宣告與呼叫、參數、`return`、函式表達式、預設值、傳值 vs 傳參考、`this`。

### 領域 3：流程控制與邏輯（3.1–3.3）
- **3.1** 評估**邏輯與比較運算式**
  → `== === != !== < > <= >=`、`&& || !`、真假值（truthy/falsy）、`Boolean()`。
- **3.2** 完成並除錯**判斷式**
  → `if / else if / else`、`switch / case / default / break`、三元運算 `? :`。
- **3.3** 完成並除錯**迴圈**
  → `for`、`while`、`do...while`、`for...of`、`break` / `continue`、巢狀迴圈。

### 領域 4：DOM 與事件（4.1–4.5）
- **4.1** 辨識與建構 **DOM 樹**
  → 節點、父/子/兄弟關係、`document` 物件、階層結構。
- **4.2** 辨識與處理 **document、表單、鍵盤、滑鼠事件**
  → `click`、`change`、`submit`、`keydown/keyup`、`mouseover`、`load`。
- **4.3** 除錯**輸出到 HTML 文件**的程式
  → `innerHTML`、`textContent`、`document.write()`。
- **4.4** 除錯**定位、修改、新增 HTML 元素**的程式
  → `getElementById`、`getElementsByClassName/TagName`、`querySelector`、`createElement`、`appendChild`、`setAttribute`、`.style`。
- **4.5** 使用**處理常式與監聽器**建立事件
  → `addEventListener("click", fn)`、`onclick` 屬性、事件物件 `event`。

### 領域 5：表單與輸入（5.1–5.3）
- **5.1** 除錯**取得表單輸入與設定欄位值**的程式
  → `input.value`、`checkbox.checked`、`select.value`、radio 群組。
- **5.2** 除錯**輸入驗證**的程式
  → 必填檢查、空值/格式判斷、數字/長度驗證、`event.preventDefault()`。
- **5.3** 描述**表單送出流程**
  → `submit` 事件、`action`/`method`、**GET vs POST**、送出前驗證。

## 五、應考者應具備的能力清單（Checklist）

- [ ] 能預測含 `+`（字串串接 vs 數字相加）、`==` vs `===`、`typeof` 的運算式結果
- [ ] 理解 `var`/`let`/`const` 差異與作用域
- [ ] 熟悉陣列常用方法（`push`/`indexOf`/`length`/`sort`）與物件屬性存取
- [ ] 會用 `Math` 常用函式與 `parseInt/Number` 轉型
- [ ] 能寫/讀 `if/else`、`switch`、`for`/`while`/`do-while`（含 `break`/`continue`）
- [ ] 能定義含參數、預設值、回傳值的函式，理解傳值/傳參考與 `this`
- [ ] 會用 `try/catch/finally` 與 `throw`；能區分語法錯誤與執行期錯誤
- [ ] 會用 `getElementById`/`querySelector` 定位元素，用 `innerHTML`/`textContent` 改內容
- [ ] 會用 `addEventListener` 綁定 `click`/`change`/`keydown` 等事件
- [ ] 會取表單欄位值（`input.value`）、做基本輸入驗證、用 `preventDefault()`
- [ ] 理解表單 `submit` 流程與 **GET vs POST** 差異
