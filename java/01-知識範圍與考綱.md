# ITS Java 證照｜知識範圍與考綱

> **證照全名**：Certiport IT Specialist (ITS) — Java
> **發行單位**：Certiport（Pearson VUE 旗下）
> **台灣總代理**：碁峰資訊
> **考試代碼**：INF-304（亦標示為 ITS-304；官方目標領域文件編號 OD-304）
> **文件用途**：完整梳理考試涵蓋的知識範圍、五大領域權重與細部技能點，作為備考的主幹地圖。
> **調研日期**：2026-07（資料以官方 Objective Domains 與多方來源交叉驗證）

---

## 一、證照定位

ITS（IT Specialist，前身為 MTA 系列的接班認證）是 Certiport 針對**入門級／基礎程度**設計的國際認證。ITS Java（考科 INF-304）用來證明應考者具備：

> 「使用 Java 撰寫、閱讀並除錯，以邏輯方式解決問題的程式碼能力。」

- **難度定位**：入門～基礎。重點在**讀懂程式、理解物件導向概念、找出錯誤**，而非在考場即席寫大型應用程式。
- **適合對象**：
  - 使用 **Java 6 SE 以上** 的應用程式開發者。
  - 軟體開發相關的中學／大專學生，或入門開發者。
  - 官方建議具備**至少 150 小時** Java 教學或實作經驗。
- **與其他證照的關係**：
  - 全部 5 個目標領域與 **CodeHS「Certiport IT Specialist Java」框架**一致。
  - 與 **OCJP / Oracle Java 認證** 定位不同；Oracle 系列偏語言與 JVM 深度，ITS 偏「基礎能力 + 國際認證背書」。
  - ⚠️ **切勿混淆**：本證照為 **Certiport ITS Java（INF-304）**，與 CodeHS 自家的 **「Java Level 1 Certification」**（90 分鐘、45 題、通過 51%、US$60）是**不同證照**，報考與選課時務必對準 Certiport 框架。

## 三、五大考試領域與權重（Objective Domains）

> ⚠️ **權重說明**：官方 Objective Domains 文件**未列各領域百分比權重**，故下表 weight 為**依官方子目標數量分布推估之近似值，非官方公告數字**。題數／時間／通過分請以官方為準。

| # | 領域（Domain） | 權重（近似） | 相對難度 |
|---|----------------|:----:|:--------:|
| 1 | Java Fundamentals（Java 基礎） | **約 20–25%** | 易 |
| 2 | Data Types, Variables, and Expressions（資料型別、變數與運算式） | **約 25–30%** | 難（占比最高） |
| 3 | Flow Control Implementation（流程控制實作） | **約 10–15%** | 中 |
| 4 | Object-Oriented Programming（物件導向程式設計） | **約 22–28%** | 難 |
| 5 | Code Compilation and Debugging（程式編譯與除錯） | **約 10–15%** | 中 |

> 📌 **備考策略提示**：領域 2（資料型別與運算式）與領域 4（物件導向）合計可達 **50% 以上**，是拉分關鍵與必守的基本盤；領域 1 概念單純、屬送分區；領域 3、5 占比中等但觀念直觀，適合穩穩拿分。

## 四、各領域細部技能點（官方子目標）

> 以下子目標對應官方 Objective Domains（OD-304）／CodeHS「Certiport IT Specialist Java」標準框架，是最權威的「考什麼」清單。

### 領域 1：Java Fundamentals（Java 基礎，1.1–1.4）
- **1.1** 描述 `main` 在 Java 應用程式中的用法
  → `main` 的簽名（`public static void main(String[] args)`）、如何以自訂類別建立實例（consume an instance of your own class）、命令列引數。
- **1.2** 使用標準套件執行基本輸入與輸出
  → 列印敘述（`System.out`）、`import` 並使用 `Scanner` 類別讀取輸入。
- **1.3** 評估變數的作用域（scope）
  → 在 block、class 或 method 內宣告變數。
- **1.4** 為程式撰寫註解與文件
  → 評估 Javadoc 的語法、撰寫語法正確的程式碼註解。

### 領域 2：Data Types, Variables, and Expressions（資料型別、變數與運算式，2.1–2.5）
- **2.1** 宣告並使用基本型別（primitive）變數
  → `byte`、`char`、`int`、`double`、`short`、`long`、`float`、`boolean`；辨識精度何時遺失；初始化；基本型別與包裝物件型別（如 `Integer`、`Boolean`）的差異。
- **2.2** 建構與分析操作字串的程式碼
  → `String` 類別與字串字面值、比較、串接、大小寫、長度；`String.format` 方法；字串運算子；字串的**不可變（immutable）**特性；初始化；`null`。
- **2.3** 建構與分析建立、迭代並操作陣列與 ArrayList 的程式碼
  → 一維與二維陣列（初始化、`null`、大小、走訪元素、存取元素）；ArrayList（新增與移除元素、遍歷）。
- **2.4** 建構與分析執行剖析、轉型與轉換的程式碼
  → 在基本型別間轉型（cast）、將基本型別轉為對應的物件型別、將字串剖析為數字、將基本型別轉為字串。
- **2.5** 建構與分析算術運算式
  → 算術運算子、指定（assignment）、複合指定運算子、運算子優先順序。

### 領域 3：Flow Control Implementation（流程控制實作，3.1–3.2）
- **3.1** 建構與分析使用**分支敘述**的程式碼
  → `if`、`else`、`else if`、`switch`；單行 vs. 區塊；巢狀；邏輯與關係運算子。
- **3.2** 建構與分析使用**迴圈**的程式碼
  → `while`、`for`、for-each、`do while`；`break` 與 `continue`；巢狀；邏輯、關係與一元（unary）運算子。

### 領域 4：Object-Oriented Programming（物件導向程式設計，4.1–4.4）
- **4.1** 建構與分析類別定義
  → 建構子、建構子多載、一個 `.java` 檔一個類別、`this` 關鍵字、基本繼承與覆寫（override）。
- **4.2** 在類別中宣告、實作並存取資料成員
  → `private`、`public`、`protected`；實例資料成員；靜態資料成員；以 `static final` 建立常數；描述封裝（encapsulation）。
- **4.3** 宣告、實作並存取方法
  → `private`、`public`、`protected`；方法參數；回傳型別；`void`；回傳值；實例方法；靜態方法；多載（overloading）。
- **4.4** 在程式中實例化並使用類別物件
  → 實例化、初始化、`null`、存取與修改資料成員、存取方法、存取與修改靜態成員、`import` 套件與類別。

### 領域 5：Code Compilation and Debugging（程式編譯與除錯，5.1–5.2）
- **5.1** 排除語法錯誤、邏輯錯誤與執行期錯誤
  → 以列印敘述除錯、`javac` 指令輸出、邏輯錯誤、主控台例外、堆疊追蹤（stack trace）判讀。
- **5.2** 實作例外處理
  → `try`、`catch`、`finally`；`Exception` 類別；例外類別型別；顯示例外資訊。

## 五、應考者應具備的能力清單（Checklist）

備考自評，能自信打勾者代表該點掌握：

- [ ] 能寫出正確的 `main` 簽名、理解命令列引數 `String[] args`
- [ ] 會用 `System.out.print/println` 輸出、會 `import` 並用 `Scanner` 讀入
- [ ] 能判斷變數在 block／class／method 中的作用域
- [ ] 熟悉 Javadoc 語法與各式程式碼註解
- [ ] 能說出各基本型別的用途，並辨識何時發生精度遺失
- [ ] 理解基本型別與包裝類別（`Integer`／`Boolean`）的差異
- [ ] 熟悉 `String` 比較／串接／`String.format`，並理解字串不可變性
- [ ] 能宣告、初始化、走訪一維與二維陣列，並操作 ArrayList
- [ ] 會在基本型別間 cast、字串與數字互轉、基本型別轉字串
- [ ] 能依運算子優先順序判斷算術與複合指定運算式的結果
- [ ] 能讀懂並預測 `if/else if/switch` 與 `for/while/for-each/do while` 的輸出
- [ ] 理解 `break`／`continue` 在巢狀迴圈中的行為
- [ ] 能撰寫含建構子（含多載）、`this`、基本繼承與覆寫的類別
- [ ] 理解 `private/public/protected`、`static final` 常數與封裝
- [ ] 能定義含參數、回傳型別與多載的實例／靜態方法
- [ ] 會實例化物件、存取與修改成員、`import` 套件與類別
- [ ] 能判讀 `javac` 輸出與 stack trace，辨識語法／邏輯／執行期錯誤
- [ ] 會用 `try/catch/finally` 攔截例外並顯示例外資訊
