# ITS Java 證照｜歷屆考題／題庫與資源彙整

> **文件用途**：彙整目前可取得的 ITS Java（考科 INF-304／ITS-304）練習題、題庫、模擬測驗與備考資源，並**標註來源與可信度**，供備考取材參考。
> **調研日期**：2026-07

## ⚠️ 先講清楚：關於「歷屆真題」

ITS（Certiport）是**商業國際認證**，官方**不公開歷屆真題**，且考題受版權與保密協議（NDA）保護。因此：

- 網路上流傳、宣稱「XX 題完整真題／100% Pass」的檔案，**多為考生回憶題或第三方模擬題**，其正確性、時效性、合法性都無法保證，且部分平台（如 dumps／真題販售站）可能涉及侵權，**不建議直接當標準答案背誦**。
- **最接近「真題手感」的合法管道**是官方授權的 **GMetrix / CertPREP / MeasureUp 模擬測驗**（見下）——它們由 Certiport 官方授權，題型、介面、難度貼近正式考試。
- 本文件將資源分成三級可信度：🟢 官方/授權 → 🟡 教材/課程 → 🟠 社群/回憶題（僅供熟悉題型，勿當標準答案）。

## 五、建議取材優先序（一句話總結）

1. **主軸**：碁峰《Java x ChatGPT 基礎必修課》中文教材 + CodeHS「Certiport IT Specialist Java」5 領域標準對齊考綱。
2. **練手感**：GMetrix / CertPREP 或 MeasureUp 官方授權模擬（至少 1–2 回全真）。
3. **補課程**：uCertify 互動課程 + 104 證照中心了解定位。
4. **自我檢測**：搭配本專案自製模擬題（原創、無版權疑慮）。
5. ❌ 避免：dumps／付費「真題解答」站（braindumps、certlibrary 等）——侵權、答案不可靠、違反考試誠信。
