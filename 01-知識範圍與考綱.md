# ITS Python 證照｜知識範圍與考綱

> **證照全名**：Certiport IT Specialist (ITS) — Python
> **發行單位**：Certiport（Pearson VUE 旗下）
> **台灣總代理**：碁峰資訊
> **文件用途**：完整梳理考試涵蓋的知識範圍、六大領域權重與細部技能點，作為備考的主幹地圖。
> **調研日期**：2026-07（資料以官方 Objective Domains 與多方來源交叉驗證）

---

## 一、證照定位

ITS（IT Specialist，前身為 MTA 系列的接班認證）是 Certiport 針對**入門級／基礎程度**設計的國際認證，用來證明應考者能夠：

> 「辨識、撰寫並除錯能以邏輯方式解決問題的 Python 程式碼。」
> （*recognize, write, and debug Python code that will logically solve a problem*）

- **難度定位**：入門～基礎。重點在**讀懂程式、理解概念、找出錯誤**，而非在考場即席寫大型程式。
- **適合對象**：高中／大專學生、程式初學者、想要一張國際入門證照豐富學習歷程或履歷者、非資訊本科但需具備 Python 基礎能力者。
- **與其他證照的關係**：
  - 上游承接自 Microsoft **98-381（MTA: Introduction to Programming Using Python）**，兩者範圍高度重疊，舊教材可參考。
  - 與 **PCEP / PCAP（Python Institute）** 定位相近但體系不同；PCEP/PCAP 更偏語言本身的深度，ITS 偏「基礎能力 + 國際認證背書」。

## 二、考試規格（Exam Format）

| 項目 | 內容 |
|------|------|
| 考試形式 | 電腦化測驗（線上／認證中心） |
| 題數 | 約 **36～43 題**（常見說法 38 題） |
| 作答時間 | **50 分鐘** |
| 滿分 / 通過分數 | 1000 分，**700 分（70%）以上通過** |
| 題型 | 單選題、複選題（多選核取）、**拖曳題**、**排序題**、配合題、**下拉式選項**、是非（yes/no）等 |
| 版本 | **Python**（selective response）／**Python Next Generation**（選擇題 + performance-based 實作題） |
| 語言 | 提供中文介面（台灣考場） |

> ⚠️ **注意版本差異**：一般常見的是選擇題版本；「Next Generation」版本加入了 performance-based（情境實作）題型，報考前請向考場確認你要考的是哪一版。

## 三、六大考試領域與權重（Objective Domains）

以下權重來自官方 Objective Domains 文件，並與 CodeHS 標準框架、GMetrix、社群讀書資源交叉驗證一致。

| # | 領域（Domain） | 權重 | 相對難度 |
|---|----------------|:----:|:--------:|
| 1 | Operations using Data Types and Operators（資料型別與運算子操作） | **20–25%** | 中 |
| 2 | Flow Control with Decisions and Loops（判斷與迴圈的流程控制） | **25–30%** | 難（占比最高） |
| 3 | Input and Output Operations（輸入與輸出操作） | **20–25%** | 易 |
| 4 | Code Documentation and Structure / Functions（程式文件、結構與函式） | **15–20%** | 中 |
| 5 | Troubleshooting and Error Handling（除錯與錯誤處理） | **5–10%** | 易 |
| 6 | Operations using Modules and Tools（模組與工具的使用） | **1–5%** | 中 |

> 📌 **備考策略提示**：領域 2（流程控制）權重最高又最難，是拉分關鍵；領域 1 與 3 合計可達 40–50%，是必守的基本盤；領域 5、6 占比低但概念單純，屬於「投報率高」的送分區。

## 四、各領域細部技能點（15 項官方子目標）

> 以下 15 項子目標對應官方 Objective Domains / CodeHS 標準框架的 15 個 standards，是最權威的「考什麼」清單。

### 領域 1：資料型別與運算子操作（1.1–1.4）
- **1.1** 評估運算式，判斷 Python 為變數指派的**資料型別**
  → `int`、`float`、`str`、`bool`、`list`、`tuple`、`dict`、`set`、`NoneType`；`type()`、動態型別、型別轉換（`int()`/`float()`/`str()`）。
- **1.2** 執行資料與資料型別的操作
  → 字串處理（切片、串接、`format`/f-string、常用方法 `.upper()/.split()/.replace()`）、串列/字典的存取與操作、索引與 slicing。
- **1.3** 依**運算子優先順序**判斷執行順序
  → 算術、比較、邏輯、指派運算子的優先級與結合性。
- **1.4** 選用正確的運算子以達成預期結果
  → 算術 `+ - * / // % **`、比較 `== != < >`、邏輯 `and/or/not`、成員 `in/not in`、身分 `is`、指派複合 `+= -=` 等。

### 領域 2：流程控制 — 判斷與迴圈（2.1–2.2）
- **2.1** 建構與分析使用**分支（branching）**的程式片段
  → `if / elif / else`、巢狀條件、布林條件組合、三元運算式。
- **2.2** 建構與分析執行**迭代（iteration）**的程式片段
  → `for`、`while`、`range()`、`break` / `continue` / `else` 子句、巢狀迴圈、遍歷字串/串列/字典。

### 領域 3：輸入與輸出操作（3.1–3.2）
- **3.1** 建構與分析**檔案 I/O**程式片段
  → `open()` 模式（`r/w/a/r+`）、`read()/readline()/readlines()/write()`、`with` 區塊、關檔。
- **3.2** 建構與分析**主控台（console）I/O**程式片段
  → `input()`（回傳字串、需型別轉換）、`print()`（`sep`、`end`、格式化輸出）。

### 領域 4：程式文件、結構與函式（4.1–4.2）
- **4.1** 為程式片段撰寫**文件**
  → 單行 `#` 註解、多行 / docstring `"""..."""`、命名與可讀性、縮排規範。
- **4.2** 建構與分析含**函式定義**的程式片段
  → `def`、參數（位置、關鍵字、預設值、`*args`/`**kwargs`）、`return`、區域/全域變數與 `scope`。

### 領域 5：除錯與錯誤處理（5.1–5.3）
- **5.1** 分析、偵測並修正有錯誤的程式片段
  → 語法錯誤（SyntaxError）、執行時錯誤、邏輯錯誤的辨識與修正。
- **5.2** 分析與建構**處理例外**的程式片段
  → `try / except / else / finally`、常見例外型別（`ValueError`、`TypeError`、`IndexError`、`KeyError`、`ZeroDivisionError`、`FileNotFoundError`）。
- **5.3** 執行**單元測試（unit testing）**
  → 測試的概念、`assert`、驗證函式輸出是否符合預期。

### 領域 6：模組與工具的使用（6.1–6.2）
- **6.1** 使用**內建模組**執行基本操作
  → `import` / `from ... import`、常見模組如 `math`、`random`、`datetime`。
- **6.2** 使用內建模組解決較複雜的運算問題
  → 綜合運用模組函式（例如 `math.sqrt`、`random.randint`、日期時間運算）解題。

## 五、應考者應具備的能力清單（Checklist）

備考自評，能自信打勾者代表該點掌握：

- [ ] 看到運算式能立刻說出結果型別與數值（含 `/` vs `//`、`%`、`**`）
- [ ] 能手算含 `and/or/not` 與比較運算子的布林運算式
- [ ] 熟悉字串索引、切片（含負索引、step）與常用字串方法
- [ ] 能讀懂並預測 `for`/`while` + `break`/`continue` 的輸出
- [ ] 理解 `range(start, stop, step)` 的邊界（不含 stop）
- [ ] 會用 `input()` 讀入並正確轉型，會用 `print()` 的 `sep`/`end`
- [ ] 能寫/讀 `with open(...) as f:` 的檔案讀寫
- [ ] 能定義含預設參數、關鍵字參數的函式，理解 return 與 scope
- [ ] 能辨識 SyntaxError / 常見執行期例外並修正
- [ ] 會用 `try/except/finally` 攔截例外
- [ ] 理解 `assert` 與單元測試概念
- [ ] 會 `import` 並使用 `math` / `random` / `datetime` 的常用函式
