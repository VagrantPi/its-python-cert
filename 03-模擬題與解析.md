## 領域 1：資料型別與運算子操作（Q1–Q6）

## 領域 2：流程控制 — 判斷與迴圈（Q7–Q13）

**Q7.** 以下程式輸出為何？
```python
x = 10
if x > 15:
    print("A")
elif x > 5:
    print("B")
else:
    print("C")
```
- (A) `A`
- (B) `B`
- (C) `C`
- (D) 無輸出

**Q8.** `for i in range(2, 10, 3): print(i, end=" ")` 的輸出為何？
- (A) `2 5 8`
- (B) `2 4 6 8`
- (C) `2 5 8 11`
- (D) `3 6 9`

**Q9.** 以下迴圈的輸出為何？
```python
total = 0
for n in range(1, 6):
    if n == 3:
        continue
    total += n
print(total)
```
- (A) `15`
- (B) `12`
- (C) `10`
- (D) `6`

**Q10.** 以下 `while` 迴圈會印出什麼？
```python
i = 0
while i < 5:
    i += 1
    if i == 3:
        break
    print(i, end=" ")
```
- (A) `1 2`
- (B) `1 2 3`
- (C) `1 2 3 4`
- (D) `0 1 2`

**Q11.**（排序題）要印出 1 到 3 的平方（`1 4 9`），請將下列片段排成正確順序：
```
① print(i ** 2, end=" ")
② for i in range(1, 4):
③（迴圈結束）
```
- (A) ② → ① → ③
- (B) ① → ② → ③
- (C) ② → ③ → ①
- (D) ① → ③ → ②

**Q12.** 巢狀迴圈輸出幾個星號？
```python
count = 0
for a in range(3):
    for b in range(2):
        count += 1
print(count)
```
- (A) `5`
- (B) `6`
- (C) `3`
- (D) `2`

**Q13.** 以下程式輸出為何？
```python
for i in range(3):
    print(i)
else:
    print("done")
```
- (A) `0 1 2`（不含 done）
- (B) `0` `1` `2` `done`（各一行）
- (C) 只印 `done`
- (D) 語法錯誤，`for` 不能接 `else`

## 領域 4：程式文件、結構與函式（Q19–Q21）

**Q19.** 以下函式的輸出為何？
```python
def greet(name, greeting="Hi"):
    return greeting + ", " + name

print(greet("Amy"))
```
- (A) `Hi, Amy`
- (B) `Amy, Hi`
- (C) `greeting, Amy`
- (D) 錯誤：缺少參數

**Q20.** 關於 Python 註解／文件字串，下列何者**錯誤**？
- (A) 單行註解以 `#` 開頭
- (B) `"""..."""` 可作為函式的 docstring
- (C) 直譯器會執行 `#` 後面的內容
- (D) 良好的縮排與命名有助於可讀性

**Q21.** 以下程式輸出為何？（變數作用域）
```python
x = 5
def f():
    x = 10
    return x
print(f(), x)
```
- (A) `10 10`
- (B) `10 5`
- (C) `5 5`
- (D) `5 10`

## 領域 5：除錯與錯誤處理（Q22–Q23）

**Q22.** 以下程式的輸出為何？
```python
try:
    print(10 / 0)
except ZeroDivisionError:
    print("cannot divide by zero")
finally:
    print("done")
```
- (A) 只有 `done`
- (B) `cannot divide by zero`
- (C) `cannot divide by zero` 換行 `done`
- (D) 程式崩潰並顯示 Traceback

**Q23.** 執行 `int("abc")` 會引發哪一種例外？
- (A) `TypeError`
- (B) `ValueError`

## 📖 解析與答案（Answer Key）

| 題 | 答案 | 解析 |
|:--:|:----:|------|
| Q1 | **B** | `/` 為真除法，結果一律為 `float`，故 `type(7/2)` 是 `float`（若用 `//` 才會是 `int`）。 |
| Q2 | **A** | `7 // 2 = 3`（整數除法向下取整）、`7 % 2 = 1`（餘數）、`2 ** 3 = 8`（次方）。 |
| Q3 | **B** | 切片 `[1:4]` 取索引 1,2,3（不含 4）：`y`,`t`,`h` → `yth`。 |
| Q4 | **C** | `*` 優先於 `+`：`"thon"*2 = "thonthon"`，再串接 `"Py"` → `Pythonthon`。 |
| Q5 | **A, B, C** | A 真且真=True；B `not False`=True；C `"a"` 在 `"banana"` 中=True；D 整數 5 與字串 "5" 不相等=False。 |
| Q6 | **C** | 先次方 `4**2=16`，再乘 `3*16=48`，再加 `2+48=50`。 |
| Q7 | **B** | `x=10`：不 >15，但 >5 → 印 `B`。 |
| Q8 | **A** | `range(2,10,3)` = 2,5,8（步長 3，不含 10）。 |
| Q9 | **B** | 1+2+4+5=12（`n==3` 被 `continue` 跳過）。 |
| Q10 | **A** | i=1 印 1、i=2 印 2、i=3 觸發 `break`（先判斷後才 print，故 3 不印）→ `1 2`。 |
| Q11 | **A** | 先寫 `for`（②），迴圈內 `print`（①），再結束（③）。 |
| Q12 | **B** | 外 3 × 內 2 = 6 次。 |
| Q13 | **B** | 迴圈正常結束（未 break）會執行 `for...else`，依序印 0、1、2，最後 `done`。 |
| Q14 | **B** | `input()` 回傳**字串** `"5"`，`"5"*2 = "55"`（字串重複）。 |
| Q15 | **B** | 需先 `int()` 轉型再運算：`int(input())*2` → `10`。A、C 皆為字串串接/重複。 |
| Q16 | **B** | `sep="-"` 以連字號分隔 → `a-b-c`。 |
| Q17 | **A, C, D** | A 對；C `with` 自動關檔；D `read()` 回傳整份字串。B 錯：`"a"` 是附加不覆蓋，覆蓋的是 `"w"`。 |
| Q18 | **C** | `"a"`（append）附加到末端且不清空；`"w"` 會清空重寫。 |
| Q19 | **A** | 使用預設參數 `greeting="Hi"` → `"Hi, Amy"`。 |
| Q20 | **C** | 錯誤敘述：`#` 後為註解，直譯器**不會**執行。 |
| Q21 | **B** | 函式內 `x=10` 為區域變數，`f()` 回傳 10；外層全域 `x` 仍為 5 → `10 5`。 |
| Q22 | **C** | 除以 0 觸發 `ZeroDivisionError`，被攔截印訊息；`finally` 一定執行印 `done`。 |
| Q23 | **B** | `"abc"` 無法轉整數 → `ValueError`。 |
| Q24 | **A** | `math.sqrt(16)=4.0`（float）、`math.floor(3.7)=3`（向下取整）。 |
| Q25 | **A** | A 對：`from math import sqrt` 後可直接呼叫。B 錯需 `math.sqrt`；C 錯 `randint(1,6)` 含 1~6 不會是 7；D 錯 import 不限第一行。 |
